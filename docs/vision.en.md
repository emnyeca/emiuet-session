# Vision

Emiuet Session is a standalone improvisation instrument designed for jazz sessions, machine‑live performance and harmonic exploration.

Unlike a traditional keyboard, Emiuet Session is aware of the current harmonic context.

A song imported from Changes provides:

- Song structure
- Chord progression
- Current position

From this information, Emiuet Session generates a Local Pitch Collection (LPC) and presents it through an expressive performance surface.

The goal is not to automate improvisation.

The goal is to help performers navigate harmonic space while retaining full artistic control.

The instrument should feel closer to a musical companion than a MIDI controller.
