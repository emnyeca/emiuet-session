# Phase 1 Testboard KiCad Project

This KiCad project records the breadboard wiring contract for the Phase 1 Teensy prototype.

It is not a production PCB yet. The first goal is to keep the test setup reviewable while firmware and sensor behavior are measured.

Files:

- `phase1-testboard.kicad_pro`
- `phase1-testboard.kicad_sch`
- `phase1-testboard.kicad_pcb`

The PCB file currently contains only a simple reference board outline and silkscreen notes. Add actual connector footprints after the first breadboard test confirms the pinout and sensor wiring.
